# Simula Campeão

Simulador interativo de chaveamento — Copa do Mundo, Libertadores e Copa do Brasil.
Monte o caminho até o título clicando nos confrontos e compartilhe por imagem ou link.

## Rodar

```bash
npm install
npm run dev
```

## Testar

```bash
npm test
```

## Build

```bash
npm run build
```

## Deploy

Site estático (Vite). Deploy na Vercel/Netlify apontando para `dist/`.
O `vercel.json` já configura o fallback de SPA.

---

Site não-oficial, feito por fãs. Sem vínculo com FIFA, CONMEBOL ou CBF.
Nomes de clubes e seleções pertencem às respectivas entidades.
